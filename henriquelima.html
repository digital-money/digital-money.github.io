O QUE É UMA ICO?

ICO, em inglês, é uma abreviatura para Initial Coin Offering, que em tradução significa Oferta Inicial de Moeda. É a fase inicial da criação de uma criptomoeda, esse mecanismo é utilizado para financiar o desenvolvimento de criptomoedas, afim de obter capital para que o projeto se torne realidade.
